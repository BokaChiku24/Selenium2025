Create Database

i) create database SeleniumData; // SeleniumData is the database name
Note: 1 row effected it notify query execute successfully. It acts as acknowledgement.
Comment: #Creating Database

ii) For use and doing operation first point to the database:
#Pointing to database
use SeleniumData;

iii) Create table under the database
#Create a table (Combination of row + column)
create table EmployeeInfo(name varchar(20), id int, location varchar (20), age int)

iv) #Table Schema or Table Structure and Constraint
describe EmployeeInfo;

Field         Type          Null   Key   Default  Extra
name          varchar(20)    Yes         NULL
id            int            Yes         NULL
location      varchar(20)    Yes         NULL
age

v) Insert the records into the table.
